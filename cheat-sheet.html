Looping in Ruby and JavaScript

This is a brief, simple cheat sheet comparing the methods of looping in Ruby and JavaScript.

Ruby

for

            
              for variable in expression do
                # code
              end

loop

            
              loop do
                # code to be executed
              end

              controlling the loop:

              n = 1
              loop do
                n = n + 1
                break if n > 9
              end

              skip the next iteration without
              finishing the current one:

              n = 1
              loop do
                n = n + 1
                next unless n == 10 break
              end

while

            
              while condition do
                # code to be executed
              end

until

            
              code until conditional

              OR

              begin
                 # code
              end until conditional

JavaScript

for

            
              for (state; condition; iterator) {
                // code block to be executed
              }

for/in

            
              var person = {fname:"John", lname:"Doe", age:25};

              var text = "";
              var x;
              for (x in person) {
                  text += person[x];
              }

while

            
              while (condition) {
                // code block to be executed
              }

do/while

            
              do {
                // code block to be executed
              }
              while (condition);

    Notes for Ruby:

  • break - terminates the most internal loop or a method with an associated block if called within the block (with the method returning nil).
  • next - jumps to next iteration of the most internal loop or terminates execution of a block if called within a block (with yield or call returning nil).

    Notes for JavaScript:

  • break - jumps out of a loop
  • continue - jumps over one iteration of the loop
